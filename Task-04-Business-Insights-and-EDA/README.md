# Task 04 - Business Insights and Exploratory Data Analysis (EDA)

## Objective

The objective of this task is to perform Exploratory Data Analysis (EDA) on a Marketing Campaign dataset, clean the data, visualize key trends, and generate business insights to support data-driven marketing decisions.

## Tools Used

- Python
- Google Colab
- Pandas
- Matplotlib

## Dataset

**Marketing Campaign Dataset**

## Steps Performed

- Imported the dataset into Google Colab.
- Explored the dataset structure and data types.
- Identified and handled missing values.
- Filled missing values in the Income column using the median.
- Converted the customer date column to datetime format.
- Performed Exploratory Data Analysis (EDA).
- Created visualizations to identify customer and marketing trends.
- Exported the cleaned dataset.

## Visualizations

- Customer Income Distribution
- Marketing Campaign Response
- Product Spending Analysis
- Customer Education Distribution

## Files Included

- Original Dataset (.csv)
- Cleaned Dataset (.csv)
- Google Colab Notebook (.ipynb)
- Screenshots
- README.md

## Outcome

Successfully cleaned the dataset, performed exploratory data analysis, created insightful visualizations, and generated business insights that can support marketing decisions.
