# Task 03 - Interactive Visualization with Power BI

## Objective

The objective of this task is to design and develop an interactive dashboard using Power BI to analyze a real-world dataset. The dashboard enables users to explore the data through interactive filters and visualizations, helping answer key business questions and support data-driven decision-making.

## Tools Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)

## Dataset

**HR Employee Attrition Dataset**

## Dashboard Features

* Interactive slicers for filtering data
* Department-wise employee analysis
* Attrition analysis by age group and gender
* Job role and salary insights
* Overall employee attrition overview
* Dynamic charts and KPIs

## Steps Performed

* Imported the HR Employee Attrition dataset into Power BI.
* Cleaned and transformed the data using Power Query.
* Created relationships (if required) and verified data quality.
* Designed an interactive dashboard with multiple visualizations.
* Added slicers for filtering by relevant fields.
* Built KPI cards and charts to highlight important insights.
* Analyzed employee attrition trends and contributing factors.

## Files Included

* Power BI Dashboard (.pbix)
* HR Employee Attrition Dataset (.csv/.xlsx)
* Dashboard Screenshots
* README.md

## Outcome

Successfully created an interactive Power BI dashboard that provides meaningful insights into employee attrition. The dashboard allows users to explore the data dynamically, identify trends, and understand factors influencing employee turnover.
