# Task 02 - Data Cleaning and Preprocessing

## Objective

The objective of this task is to clean and preprocess the Superstore Sales dataset using Python and Pandas to improve data quality before analysis.

## Tools Used

* Python
* Pandas
* Google Colab

## Dataset

Sample - Superstore Sales Dataset

## Steps Performed

* Loaded the dataset into Google Colab.
* Explored the dataset using `df.head()`.
* Checked for missing values.
* Filled missing values using the median.
* Checked for duplicate records.
* Removed duplicate rows (if any).
* Converted the **Order Date** and **Ship Date** columns to datetime format.
* Saved the cleaned dataset as a CSV file.

## Files Included

* `Task2_Data_Cleaning.ipynb`
* `Sample - Superstore Sales (Excel).xls`
* `Cleaned_Superstore_Data.csv`
* Screenshots of each preprocessing step

## Outcome

The dataset was successfully cleaned and prepared for further data analysis by handling missing values, checking duplicate records, formatting date columns correctly, and exporting the cleaned dataset.
